Security and compliance at Dialog

Information security

Architecture

Dialog is available as On-premise or Private Cloud messenger. Unlike public cloud solutions, Dialog LLC does not have access to the client’s infrastructure. This architecture offers the best protection against intruders.

High security standards

Data encryption

Protection of client-server connections using TLS and DTLS protocols.



DLP

Dialog integrates with SIEM systems and allows you to respond to security events before damage occurs.

Security audit

Dialog cooperates with leading security experts to conduct regular audits of its products.



SIEM

Integrates with data leak prevention systems via ICAP. The risk of intentional or accidental information leaks is minimized.

Integration with antivirus software

Support for integration with antiviruses via ICAP. The messenger blocks malicious files when sending.
