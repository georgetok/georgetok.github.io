Безопасность и соответствие требованиям в Dialog

Защита информации

Архитектура

В отличие от облачных решений Dialog не имеет ключей для расшифровки вашего общения и доступа к инфраструктуре. Эта архитектура предлагает лучшую защиту от злоумышленника.

Высокие стандарты безопасности

Шифрование передачи данных

Защита соединений «клиент-сервер» по протоколам TLS и DTLS.



DLP

Интегрируется с системами предотвращения утечки данных по протоколу ICAP. Минимизирован риск умышленной или случайной утечки информации.

Интеграция с антивирусным ПО

Поддержка интеграций с антивирусами по протоколу ICAP. Мессенджер блокирует вредоносные файлы при отправке.



SIEM

Dialog интегрируется с SIEM-системами и позволяет реагировать на события безопасности до наступления ущерба.

Аудит безопасности

Сотрудничество с ведущими экспертами по безопасности для проведения регулярных аудитов своих продуктов.
