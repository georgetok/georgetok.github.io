Dialog

Корпоративный мессенджер для совместной работы и безопасного общения

Преимущества приложения Dialog

  • Единая коммуникационная среда

    Легко находите коллег, общайтесь в групповых чатах, обменивайтесь файлами и совершайте аудио- и видеозвонки.

  • Сокращение расходов на коммуникации

    Объединение разрозненных инструментов в одном решении позволяет сократить расходы на ПО, оборудование, обслуживание и поддержку

  • Защищенная коммуникация

    Архитектура enterprise уровня, интеграция с системами корпоративной безопасности.

Функции Dialog

Групповые чаты и каналы

Создавайте чаты для команд и проектов – фокусируйте усилия, делайте работу более прозрачной, а информацию — доступной. Используйте каналы для централизованного информирования сотрудников.

Обмен файлами

Тяжелая презентация не отправляется по почте? Делитесь файлами, не задумываясь о лимитах по размеру или формату.

Аудио- / видеозвонки

Созванивайтесь с коллегам в мессенджере или подключайтесь к конференциям в АКС/ВКС-системе.

Поиск

За считанные секунды находите необходимые файлы, изображения, документы, а благодаря интеграции с адресной книгой – нужного сотрудника по имени, должности или отделу.

Чат-боты

Создавайте ботов и чат-ботов для автоматизации бизнес-процессов с помощью Dialog Bot SDK.

Решения

  • Бизнесу

    Ведущие российские компании повысили эффективность совместной работы, объединив с помощью Dialog тысячи сотрудников и десятки корпоративных систем.

  • Государственным организациям

    Dialog обеспечивает надежную и безопасную коммуникацию в государственных структурах. Мессенджер включен в Реестр российского ПО и разрабатывается в соответствии с требованиями Импортозамещения. Разработчик продукта имеет лицензии ФСТЭК и ФСБ России.

Dialog помогает командам стать эффективнее

Создавайте интеграции со сторонними приложениями и автоматизируйте рутинные процессы, используя Dialog Bot SDK и Dialog API.

Dialog Bot SDK

Bot SDK позволяет создавать ботов и чат-ботов для Dialog. Боты оптимизируют повторяющиеся рутинные операции и освобождают время для более сложных задач. SDK доступен для JS, Java и Python.

Реализовано с помощью Bot SDK

Помощник HR bot

Упрощает работу с документами: опрашивает сотрудника по ключевым вопросам и формирует заявление на отпуск.

Dialog API

API позволяет другим приложениям взаимодействовать и обмениваться данными с Dialog.

Реализовано с помощью Dialog API

Совместная работа над документами МойОфис

Возможность автоматически создавать группы для обсуждения, уведомлять о предоставлении доступа и отправлять ссылки на документы в Dialog, не выходя из офисных приложений МойОфис.

Dialog в большом бизнесе

Сбербанк запустил корпоративный мессенджер СберЧат на базе Dialog для более чем 300 000 сотрудников.

Оперативное взаимодействие и обмен данными важны для компаний любого размера и профиля деятельности. При этом банку и компаниям экосистемы было необходимо собственное масштабируемое решение, обеспечивающее эффективную и защищенную коммуникацию большому количеству сотрудников. Dialog реализовал решение, которое позволит не только оптимизировать процесс коммуникации, но и повысить эффективность совместной работы за счет создания единого цифрового рабочего пространства.
